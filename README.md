# Task-2-IbrahimAdekunle

## Project Overview

This project presents an exploratory data analysis (EDA) of an e-commerce sales dataset using Microsoft Excel. The analysis was conducted to identify patterns, trends, and relationships within the data and generate meaningful business insights through descriptive statistics and data visualisation.

## Project Objectives

* Summarise key numerical variables.
* Analyse product revenue performance.
* Evaluate payment method preferences.
* Examine the distribution of order statuses.
* Analyse monthly sales trends.
* Assess referral source contributions.
* Generate business insights to support decision-making.

## Exploratory Data Analysis Activities

* Performed a statistical overview of key numerical variables.
* Conducted product revenue analysis.
* Analysed payment method distribution.
* Examined order status distribution.
* Investigated monthly sales trends.
* Evaluated referral source performance.
* Created PivotTables and PivotCharts for data visualisation.
* Interpreted findings and generated business insights.

## Tools and Features Used

* Microsoft Excel
* PivotTables
* PivotCharts
* Descriptive Statistics
* Sorting and Filtering
* Data Visualisation

## Repository Contents

* 📊 [EDA Workbook](./EDA_Workbook.xlsx) Excel workbook containing the complete exploratory data analysis, including PivotTables, PivotCharts, visualisations, and business insights.

## Key Findings

* Product revenue was distributed across multiple product categories.
* Payment methods exhibited relatively balanced customer preferences.
* Order statuses were evenly distributed across transactions.
* Monthly sales displayed noticeable fluctuations throughout the year.
* Referral sources contributed to customer acquisition through diverse marketing channels.
